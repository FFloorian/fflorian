Guide DNS

Publié le 21 juillet 2025

✅ Pourquoi changer de DNS ?

  • Vie privée : Choisissez un fournisseur qui ne conserve pas vos données.
  • Filtrage : Bloquez publicités, trackers ou contenus adultes.
  • Restrictions : Contourner les blocages mis en place par votre FAI.

🖥️ 1. Changer DNS sur Windows 10 / 11

  1. Clique droit sur l’icône réseau dans la barre des tâches (en bas à droite).
  2. Choisis « Ouvrir les paramètres Réseau et Internet ».
  3. Dans la fenêtre qui s’ouvre, clique sur « Modifier les options d’adaptateur » (panneau de droite).
  4. Clique droit sur ta connexion active (Ethernet ou Wi-Fi) puis choisis « Propriétés ».
  5. Double-clique sur « Protocole Internet version 4 (TCP/IPv4) ».
  6. Coche « Utiliser l’adresse de serveur DNS suivante ».
  7. Entre les DNS que tu souhaites (ex. Cloudflare : préféré 1.1.1.1, auxiliaire 1.0.0.1).
  8. Clique sur « OK » puis encore « OK » pour fermer les fenêtres.

🍏 2. Changer DNS sur macOS (Ventura+)

  1. Réglages Système → Réseau → Wi-Fi → Avancé → DNS.
  2. Ajoutez vos serveurs DNS (ex. Cloudflare : 1.1.1.1, 1.0.0.1).

🌐 3. Navigateurs

Firefox

  1. Paramètres → Vie privée & sécurité → Activer DNS via HTTPS.
  2. Saisissez l’URL DoH (ex : https://dns.mullvad.net/dns-query).

Chrome / Edge / Brave

  1. Paramètres → Confidentialité et sécurité → DNS sécurisé → Personnalisé.
  2. Ajoutez votre URL DoH (Cloudflare, Mullvad, etc.).
